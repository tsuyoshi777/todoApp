# TodoApp
Todoアプリの作成

## 必須機能
- [ ] タスクが一覧で表示されること
- [ ] タスクを文字列で入力できること
- [ ] タスクの登録ができること
- [ ] 最低2画面用意し画面線画できること

## 作成条件
- [ ] プロジェクトの新規作成から行う
- [ ] タスクのデータに関してWeb APIを使い管理
- [ ] Web APIはRest、データ形式はJSONとして別途APIインターフェース仕様を用意する
